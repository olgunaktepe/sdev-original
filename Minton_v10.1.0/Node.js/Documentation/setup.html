We utilize Gulp to automate our build flow, allowing for complete automation of our development workflows. Gulp is a powerful toolkit designed to help developers eliminate time-consuming tasks, allowing them to focus on what really matters: building great products. If you're not familiar with Gulp, you can learn more about it here.

Prerequisites

To get started with Minton, please follow the steps below to install and set up all the necessary prerequisites:

  • Node.js

    To use our build tools and take full advantage of Minton's customization options, you will need to download and install Node.js. If you do not have Node.js v16.x or v18.x installed already, you can download the package installer from the official website. We recommend that you download the stable version of Node.js (LTS) to ensure the best possible experience. Download Node.js

  • Gulp

    Ensure that you have Gulp installed and running on your computer. If you already have Gulp installed, you can skip this step. If not, you can install it by running the following command from your terminal: npm install -g gulp@4.0.1.

Installation

You can run the following commands to run projects locally: Or build for production use:

1. Bun (Recommended for mac and Linux OS)

If you don't have bun installed on your PC, use the npm i -g bun or sudo npm i -g bun to install

Command Description
bun i This would install all required dependencies node_modules folder..
gulp This would compile all the resources to public folder.
bun dev Runs the project locally, starts the vite and node development server.
gulp build It bundles with production mode. Your app is now ready to be deployed.
bun preview It boots up a local static web server that serves the files from dist.
2. Yarn

If you don't have yarn installed on your PC, use the yarn i -g bun or sudo npm i -g yarn to install

Command Description
yarn This would install all required dependencies node_modules folder..
gulp This would compile all the resources to public folder.
yarn dev Runs the project locally, starts the vite and node development server.
gulp build It bundles with production mode. Your app is now ready to be deployed.
yarn preview It boots up a local static web server that serves the files from dist.
3. NPM

npm comes preinstalled when you install Nodejs

Command Description
npm i or npm i --force This would install all required dependencies node_modules folder..
gulp This would compile all the resources to public folder.
npm run dev Runs the project locally, starts the vite and node development server.
gulp build It bundles with production mode. Your app is now ready to be deployed.
