Installation

Prerequisites

Please follow below steps to install and setup all prerequisites:

Python

Make sure to have the Python installed & running in your computer. If you already have installed Python on your computer, you can skip this step. Please use Python version 3

For Windows
  • Download python from Microsoft store
For Linux and macOS
  • sudo apt update
  • sudo apt install python3
Virtualenv

Make sure to have the virtualenv installed globally & running on your computer. If you already have installed on your computer, you can skip this step.

For Windows
  • py -m pip install --user virtualenv
For Linux and macOS
  • python3 -m pip install --user virtualenv
Installation

To setup, follow the below mentioned steps:

Command Description
python3 -m venv environment_name
 Create Virtual Environment on Linux & macOS
python -m venv environment_name
 Create Virtual Environment on Windows OS
source environment_name/bin/activate
 Activate Environment on Linux & macOS
environment_name/Scripts/activate
 Activate Environment on Windows OS
pip install -r requirements/local.txt
 This would install all the required python dependencies.
python3 manage.py migrate
 Run the migration to sync the database on Linux & macOS
python manage.py migrate
 Run the migration to sync the database on Windows OS
python3 manage.py runserver
 Starts the local server on localhost:8000 on Linux & macOS
python manage.py runserver
 Starts the local server on localhost:8000 on Windows OS
Note: If You want to use SCSS then follow mentioned steps.
Prerequisites

Please follow below steps to install and setup all prerequisites:

Node.js

In order to use build tools you will need to download and install Node.js.

Gulp

Make sure to have the Gulp installed & running in your computer. If you already have installed gulp on your computer, you can skip this step. In order to install, just run command npm install -g gulp from your terminal.

Installation

To setup, follow the below mentioned steps:

You can run this app using any following package managers:

  • Bun: npm i -g bun or sudo npm i -g bun
  • Yarn: npm i -g yarn or sudo npm i -g yarn
  • NPM: comes pre-installed with Nodejs
Command Description
npm i or yarn or bun i This would install all required dependencies in node_modules folder.
gulp It bundles assets and runs the project locally, starts the development server.
gulp build It bundles assets into the static directory. Your app is now ready to be deployed.
We recommend to run this command before starting django server.

Note:

  • During Development, keep the gulp running and then run the python's runserver command in a separate terminal
  • For Production, create a production build with above build commands and then run the python's runserver command
