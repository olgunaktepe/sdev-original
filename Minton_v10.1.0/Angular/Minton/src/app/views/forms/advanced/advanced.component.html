Selectize

Selectize is the hybrid of a textbox and <select> box. It's jQuery-based and it's useful for tagging, contact lists, country selectors, and so on.


Switchery

iOS 7 style switches for your checkboxes

use ui-switch element and Add an attribute color="@colors" to your input element and it will be converted into switch.

Add an attribute size="small",size="large" to your input element and it will be converted into switch. Add an attribute color="@color" defaultBgColor="@color" to your input element and it will be converted into switch.
