Routing

Minton Angular is having routing setup based on Angular-Router.

The pre configured routing would make your job easy when dealing with any routing related changes. The routing is setup in a way to support lazy loading as well. You can easily add, change or remove any route by simply making changes described below:

  1. Enable routing in the project using --routing while creating project.

  2. Create new file your-module.routes.ts, declare path and config for your components inside new your module.

  3. Add the entry for same in main application routing src/app/app.routes.ts E.g.

    { path: 'dashboard1', component: MainLayoutComponent, loadChildren: () => import('./views/views.route').then((mod) => mod.VIEW_ROUTES), canActivate: [ (url: any) => { const router = inject(Router) const authService = inject(AuthenticationService) if (!authService.session) { return router.createUrlTree(['/auth/login'], { queryParams: { returnUrl: url._routerState.url }, }) } return true }, ] }

    Each of these properties are explained below:

    • path - relative path to component
    • loadChildren - module to load on this route
    • canActivate - set of Guards which decides whether route can be actiavted or not. It restrict the non authenticated user to access this module.
Note that you don't need to restart the development server in order to see the menu changes getting in effect

Optionally, you might want to add the menu entry in respective layout menu component.
