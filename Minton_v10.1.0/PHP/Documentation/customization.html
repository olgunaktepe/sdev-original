Customization

In-built Demos (Skins)

Minton PHP comes with multiple demos (skins). I.e. 'corporate', 'creative', 'default', 'material', 'modern', 'saas'. Each of these skins are having respective dark mode available as well. You can easily use any of them by simply changing the css file reference in your file.

In your file, simply change the reference of **.min.css and **-dark.min with respective demo (skin) css file and you would have it activated.

In-built Layouts

Minton PHP provides multiple choices when it comes to layouting. There are multiple layout choices available. I.e. Vertical (main navigation on "Left") and Horizontal (main navigation on "Top"). You can easily use any of them by simply changing the few partials and using data attributes on html element.

Check out the pages layouts-horizontal.php files available to see how the respective layout can be activated.

Customizing Color Mode, Left Sidebar, Topbar, Layout Width and Right Sidebar

Minton allows you to have customized left sidebar, overall layout width or right sidebar. You could turn the left sidebar to different size, theme (look) and size. You can customize it by specifying the layout data attribute (data-layout={}) on html element in your file. The config object properties accept the following values:

Title Type Options Description
data-bs-theme String "dark" | "light" Changes overall color scheme to dark
data-layout-width String "fluid" | "boxed" Changes the Content width of overall layout
data-layout-position String "fixed" | "scrollable" Changes the Content position of overall layout
data-topbar-color String "dark" | "light" | "brand" Changes topbar color scheme to dark or light
data-menu-color String "light" | "dark" | "brand" | "gradient" the Menu color scheme. By default, it would be set to default (light)
data-sidebar-size String "default" | "compact" | "condensed" Changes overall Sidebar Width.
(* Only available in Sidebar Menu)
data-sidebar-user Boolean "true" | "false" Indicates whether to show Sidebar on opening up the page.
(* Only available in Sidebar Menu)
data-layout-mode String "default" | "horizontal" | "two-column" | "detached" Changes overall layout mode
(Manually) (* Available in Sidebar & Tow Colum Sidebar Menu)
data-two-column-color String "dark" | "brand" | "gradient" the Two Colum Icon Menu color scheme. By default, it would be set to default (light)
(Manually Add) (* Only available in Two Column Sidebar Menu)

Adding new page

We have provided a starter page (check pages-starter.php). It allows you to get started easily and helps you to add new page. Please note following important points:

  • Make sure you have included assets/css/bootstrap.min.css, assets/css/app.min.css, assets/css/icons.min.css, or you can include partials/head-css.php file and also include assets/js/vendor.min.js , assets/js/app.min.js in your code.
    If you would like to have dark mode, make sure to include the assets/css/app-dark.min.css.
  • Most of default/basic form elements along with few advanced elements are available and bundled in above css and js and so you don't need to include any css or js separately.
  • Few elements e.g. charts, data tables, calendar, maps etc requires you to include corresponding css and js files in your code. Please check corresponding documentation page for the same.
